Home Lab 001-Ubuntu VM Setup

Objective:
Build the foundation of cybersecurity home lab using VMware and Ubuntu.

VM Environment
- Host: MacBook Pro
- VMware Fusion
- Ubuntu Desktop
- 4 GB RAM
- 2 CPU cores
- 50 GB Storage

Tasks Completed
- Installed VMware Fusion
- Created Ubuntu VM
- Installed Ubuntu Desktop
- Verified system functionality
- Created and tested VM snapshots

Lessons Learned
- Learned how virtualization works
- Learned the purpose of snapshots
- Verified that the snapshots taken can restore a system to previous state
